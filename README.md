# Amazon Clone - Portfolio Project

This project is a visual clone of the Amazon homepage, built for **educational and portfolio purposes only**.

## Project Overview
* **Purpose:** This project was developed to practice front-end web development skills, specifically focusing on CSS layout and HTML structure.
* **Status:** This is a non-commercial, static demonstration site. It is not affiliated with, endorsed by, or in any way officially connected to Amazon.com.
* **Functionality:** This site does not collect user data, process payments.

## Technologies Used
* HTML
* CSS
